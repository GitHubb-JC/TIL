# 리스트 (List)

## 1. 배열 vs 연결리스트

### 1) 배열 (Array)

- 여러 데이터들이 연속된 메모리 공간에 저장되어 있는 자료구조

### 2) 연결 리스트 (Linked List)

- 데이터가 담긴 여러 노드들이 순차적으로 연결된 형태의 자료구조



## 2. 파이썬의 리스트

### 1) .append(원소)

- 리스트 맨 끝에 새로운 원소 **삽입**
- return 값 없어

### 2) .pop(인덱스)

- 특정 인덱스에 있는 원소를 **삭제 및 반환**
- return 값 있어

### 3) .count(원소)

- 리스트에서 해당 원소의 **개수**를 반환

### 4) .index(원소)

- 리스트에서 처음으로 원소가 등장하는 **인덱스** 반환
- 최악의 상황에서 순회니까 O(n)

### 5) .sort()

- 리스트를 오름차순으로 정렬
  reverse = True 옵션을 통해 내림차순으로 정렬 가능
